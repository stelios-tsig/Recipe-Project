# 🍳 Kitchen Recipes App

Εφαρμογή διαχείρισης συνταγών μαγειρικής σε Python με GUI (customtkinter).

## Περιγραφή

Ο σκοπός του προγράμματος είναι η ανάρτηση καθώς και η επιτυχής ακολουθία συνταγών. 
Κάθε χρήστης μπορεί να ανεβάσει ή να ακολουθήσει όσες συνταγές θέλει. 
Περιέχει οθόνη με συγκεκριμένες γνωστές κατηγορίες για άνετη αναζήτηση.

## Λειτουργίες

- **Αναζήτηση Συνταγών**: Εμφάνιση λίστας με όλες τις συνταγές
- **Προσθήκη Συνταγής**: Δημιουργία νέας συνταγής με υλικά και βήματα
- **Ακολούθηση Συνταγής**: Λεπτομερής προβολή με checklist υλικών και οδηγίες βήμα-βήμα
- **Κατηγορίες**: Ψάρια, Ζυμαρικά, Κρέας, Φούρνος, Air Fryer, Vegan

## Εγκατάσταση

```bash
pip install -r requirements.txt
```

## Εκτέλεση

```bash
python ui.py
```

## Αρχεία

- `ui.py` - Κύριο αρχείο εφαρμογής (GUI)
- `Model.py` - Μοντέλα δεδομένων (Recipe, Step, Ingredient, RecipeManager)
- `Database.py` - Διαχείριση SQLite βάσης δεδομένων
- `requirements.txt` - Εξαρτήσεις
- `README.md` - Αυτό το αρχείο
